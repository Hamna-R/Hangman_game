# Hangman_game
A simple console-based Hangman game built using Python.
## Features
- Random word generation
- Word hints system
- Available letters tracking
- Score calculation
- Input validation
- Random 3–6 letter word selection

## Technologies Used
- Python
- Random module
- String module

## How to Run

1. Download the project files
2. Make sure `words.txt` is in the same folder
3. Run the program:

```bash
python hangman.py
